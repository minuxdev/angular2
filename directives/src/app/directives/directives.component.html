Directives

Are instructions passed to HTML template to manipulate the DOM. With directives is possible to hide or show HTML Elements under centrain condition, change its properties, loop thrugh an array and render N times an element according to array length, etc. There are basically two types of directives:

  1. Structurals

    Used to modify HTML structure

    • *ngFor="let item of collection"
    • *ngIf="Property"
    • [ngSwitch]="conditionExpression", etc.

  2. Attributes

    Used to modify HTML Element styles

    • [ngClass]="{{ '{"class-name1": condition1, "class-name2": condition2}' }}"
    • [ngStyle]="{{ '{key1: value1, key2: value2}' }}"

Structural Directives

  • *ngIf="Property"

    Will render HTML Element host only if the conditionExpression evaluate true

    Since mess with the DOM can impact on performance, property bind can be used wether to hide or show element. {{ '[hidden]="Property"' }} can be used instead of {{ '*ngIf="Property"' }}, however, using [hidden] brings up security issues because hidden HTML Elements can still be accessed thrugh Developer Tools. Being so, it's extremelly important to know when to use it.
  • *ngFor="let item of collection"

    Will render HTML Element host for N collection.length

  • [ngSwitch]="conditionExpression"

    Used to choose one among a group of options. To test the conditionExpression for each possible value, *ngSwitchCase="'option'" is used and *ngSwitchDefault="default value" for default value.

Attribute Directives

  • [ngStyle]="{{ '{property1: condition, property2: condition}' }}

    Used to change or add CSS properties directly from HTML Element

  • [ngClass]="{{ '{"class-name", condition, "class-name2": condition2}' }}"

    Used to apply CSS classes to HTML Element under certain conditions. Its usage may vary depending on the type of expression evaluation.

    • string - the CSS classes listed in the string (space delimited) are added,

      [ngClass]="'class1 class2'"

    • Array - the CSS classes declared as Array elements are added,

      [ngClass]="['class1', 'class2', '...']"

    • Object - keys are CSS classes that get added when the expression given in the value evaluates to a truthy value, otherwise they are removed.

      [ngClass]="{{ '{"class1 class2": condition}' }}"
